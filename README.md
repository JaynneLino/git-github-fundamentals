# Git & GitHub Fundamentals 🛠️

This repository contains my practice notes and exercises from the **"Git and GitHub from Basics to Advanced"** course. 

### 🧠 Skills Practiced:
- **Local Workflow:** `git init`, `add`, `commit`, `status`, `log`.
- **Remote Workflow:** `push`, `pull`, `fetch`, `clone`.
- **Branching:** Creating, switching, and merging branches.
- **Collaboration:** Handling Pull Requests and Merge Conflicts.
- **Best Practices:** Writing clear commit messages and using `.gitignore`.
